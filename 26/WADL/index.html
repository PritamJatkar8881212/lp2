PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I

    Basic calculus and differential equations.

  • Engineering Physics

    Concepts of mechanics, optics, and materials.

  • Basic Electrical Engineering

    Introduction to electric circuits and devices.

Second Year (SE)

  • Data Structures

    Stacks, queues, trees, and graphs.

  • Digital Electronics

    Logic gates, flip-flops, and digital circuits.

  • Computer Networks

    OSI model, protocols, and network devices.

Third Year (TE)

  • Operating Systems

    Process management, memory and file systems.

  • Database Management Systems

    SQL, normalization, transactions.

  • Software Engineering

    SDLC models, requirements, testing.

Final Year (BE)

  • Machine Learning

    Supervised, unsupervised learning, algorithms.

  • Cloud Computing

    Virtualization, cloud services, and models.

  • Project Work

    Final year development or research-based project.
